Terminal-based taamagochi like game/simulation

Basic features:
- [ ] There is a creature that is displayed in the terminal
- [ ] The creature can move freely inside the terminal window
- [ ] The creature sometime moves to random points
- [ ] Sometimes food spawns inside the terminal
- [ ] Creature goes and eats food that has spawned
- [ ] Sometimes creature goes to sleep and wakes up
- [ ] State of the simulation can be saved to a file and loaded from it


Additional ideas:
- [ ] Creature has needs: Hunger, Sleep, Fun
- [ ] Creature statistics are displayed in the terminal
- [ ] Creature can say things using a LLM running locally
- [ ] Creature has more complex moving pattern (not only in straight lines)
