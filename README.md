# Aquarium in your terminal

Just some fish and shrimp in your terminal.

#### 1. Clone this repository
```shell
git clone https://github.com/hahalek/terminal_alien
cd terminal_alien
```

### Prerequisites:
- Blessed
```shell
pip install blessed
```

#### 1. Run with:
```shell
python3 src/main.py 
```

# Featuring:
- shrimp
- fish

# What do they do
- they swim
- press spacebar to feed them (they eat food)
- pretty much they just exist
